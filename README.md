## Description  
This project performs an end-to-end analysis of employee performance, tenure, and attrition using synthetic SuccessFactors HR data. The aim was to demonstrate the full data lifecycle—from business problem definition, data preparation, and exploratory analysis, to actionable insights and recommendations. Using Python, SQL, and Tableau, the dataset was cleaned, transformed, and integrated to uncover patterns in performance distribution, retention risks, and departmental biases. The insights inform evidence-based talent management, retention strategies, and succession planning.

### Technologies: PostgreeSQL/ Python / SQL / Tableau 
### Python Libraries: Pandas, NumPy, Matplotlib, Seaborn 

## Objectives  
* Identify trends in employee performance by department and tenure  
* Understand attrition patterns and high-risk employee segments  
* Integrate performance, tenure, and attrition data for evidence-based decision-making  
* Develop interactive dashboards for HR leaders to monitor talent and retention risks  
* Deliver actionable recommendations to improve workforce planning and succession strategies  

## Data Analyst Skills Demonstrated  
* **Data Acquisition & Integration:**  
  - Loaded synthetic employee data (CSV) and behavioural data (PostgreSQL)  
  - Integrated datasets using Employee ID to enable cross-functional analysis  
* **Data Cleaning & Preprocessing:**  
  - Validated referential integrity between datasets  
  - Checked for duplicates, missing values, and inconsistent rating scales  
  - Standardized column names and data types for accurate analysis  
* **Exploratory Data Analysis (EDA):**  
  - Examined performance distribution by department  
  - Analysed high- and low-performer retention across tenure bands  
  - Studied attrition and exit type trends based on performance ratings  
* **Insight Generation & Recommendations:**  
  - Identified department-level rating inconsistencies and potential biases  
  - Highlighted retention risks among mid-tenured high performers and low performers  
  - Suggested early intervention programs, structured PIPs, and career progression initiatives  
  - Recommended standardized performance rating calibration across departments  
* **Data Visualization & Dashboards:**  
  - Developed a Tableau executive dashboard to track performance, tenure, and attrition in real time  
  - Visualized departmental performance, tenure-based trends, attrition risk, and high/low performer segmentation  
  - Enabled proactive decision-making for promotions, succession planning, and retention strategies  

## Key Outcomes  
* Enabled data-driven, objective performance evaluations  
* Reduced bias and improved fairness across departments  
* Provided early intervention strategies for low performers  
* Highlighted retention risks and actionable steps for high performers  
* Delivered insights to support HR leadership in workforce planning, succession, and talent strategy
